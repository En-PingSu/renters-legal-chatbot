3. RAG Query Pipeline

Traces the ask() function in src/rag/pipeline.py. A user question flows through retriever selection, context assembly, LLM generation, and citation verification.

flowchart TD
  Q["User Question"]

  Q --> SEL{"Retriever Selection
RETRIEVER_REGISTRY"}

  SEL -->|"retrieve_fn(query, top_k)"| CHUNKS["Retrieved Chunks
top_k = 5 or 10"]

  CHUNKS --> FMT["format_context()
[Source N: title - url]
chunk content"]

  FMT --> PROMPT["System Prompt Assembly
5 Rules:
1. Use ONLY retrieved context
2. Never provide legal advice
3. Always cite sources
4. Disclose insufficient context
5. Stay in MA tenant law scope"]

  PROMPT --> API["OpenRouter API
temp = 0.2, max_tokens = 1500"]

  subgraph MODELS["Generator Models"]
    direction TB
    GPT["openai/gpt-4o"]
    LLAMA["meta-llama/llama-3.3-70b-instruct"]
    MISTRAL["mistralai/mistral-small-3.1-24b-instruct"]
  end

  MODELS -.-> API

  API --> VER["verify_citations()
Deduplicate source URLs
Append sources footer"]

  VER --> OUT["Response Dict
question, model, retriever,
retrieved_chunks, response"]

  CHROMA[("ChromaDB
967 chunks")] -.->|"vector search"| SEL

4. Retriever Strategy Hierarchy

Eight retriever strategies build on each other through composition. Base retrievers (Vector, BM25) are combined into increasingly sophisticated strategies. Labels match RETRIEVER_REGISTRY keys.

flowchart BT
  VEC["vector
ChromaDB cosine similarity
all-MiniLM-L6-v2"]
  BM25["bm25
Lexical search
Snowball stemmer"]

  VEC --> HYBRID
  BM25 --> HYBRID
  HYBRID["hybrid
0.6 vector + 0.4 BM25
score normalization & fusion"]

  HYBRID -->|"2x candidates"| RERANK["rerank
ms-marco-MiniLM-L-6-v2
cross-encoder scoring"]

  VEC -->|"2x candidates"| PC["parent_child
Neighbor +/-1 expansion
1.2x distance dampening"]

  HYBRID --> HPC["hybrid_parent_child
Hybrid base +
neighbor expansion"]
  HPC --> HPCR["hybrid_parent_child_rerank
Hybrid PC candidates
+ cross-encoder rerank"]

  HYBRID -->|"4x candidates"| AM["auto_merge
Merge docs with >=40%
chunks retrieved, then rerank"]

  MQ["multiquery
LLM generates 3 query variants
RRF merge, k=60"]
  MQ -->|"uses base retriever"| RERANK
